# Branch-Prediction-Archive
Here you will find documents regarding branch prediction.

## How to contribute
If it is wished to contribute to the archive by publishing documents, the following rules shall be followed:
- The document must be provided using the PDF file format.
- The name of the file that contains the document must be the same title of that document. In cases the title contains a ':' symbol, this symbol must be replaced with a '.' symbol.
- The document must be related to branch prediction.
- The document must not be a duplicate.
